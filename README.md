# Breather

Breather is an Android mobile health application designed to deliver real-time wellness feedback using biosensor data. The application analyzes incoming biometric information and provides users with responsive feedback within 20 seconds to support awareness of personal wellness trends and improve user engagement with health monitoring.

## Features

* Real-time wellness feedback using biosensor input
* Mobile-first Android application built in Android Studio
* Responsive UI focused on accessibility and ease of use
* User testing and iterative refinement to improve reliability and responsiveness
* Fast processing pipeline with feedback delivered in under 20 seconds

## Tech Stack

* Kotlin
* Java
* Android Studio

## Project Goals

The goal of Breather was to explore how mobile technologies and biosensor integration could support accessible, real-time wellness monitoring. The project focused on balancing responsiveness, usability, and reliability while incorporating user feedback throughout development.
